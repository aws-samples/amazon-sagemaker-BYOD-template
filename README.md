# SageMaker BYOD template and tutorial

## Prerequisites

- Install Docker
- Install node 8>
- Install aws cli with credentials

## setup

- install dependencies
```shell 
npm install
```
- copy config
```shell
cp config.js.example config.js
```

- edit config.js
